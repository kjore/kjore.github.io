什么是MCP

Model Context Protocol（模型上下文协议）是一种用于定义和管理AI模型上下文的协议。为大语言模型提供统一的标准化方式与外部数据源和工具之间进行通信。

MCP的作用

MCP的主要作用包括：

  1. 标准化接口：MCP定义了一套标准化的接口，使得不同的AI模型可以一致地访问和使用外部数据源和工具。
  2. 上下文管理：MCP帮助AI模型管理其上下文信息，包括用户输入、历史对话记录和外部数据源的状态。
  3. 扩展性：通过MCP，AI模型可以轻松地集成新的数据源和工具，提升其功能和性能。
  4. 互操作性：MCP促进了不同AI模型之间的互操作性，使得它们可以共享上下文信息和数据。
    借助 MCP 服务器，您可以扩展 Gemini CLI 的功能，实现超越其内置特性的操作，例如与数据库、API、自定义脚本或专业工作流进行交互。

MCP通信方式

  1. 用在本地服务，操作本地软件或者本地文件，没有在线服务
  2. 用在在线服务，操作在线API或者在线文件，有在线服务，比如调用天气API，调用股票API等
